# Lumis Studio — Brand Identity

## Core Identity
- **Name:** Lumis Studio
- **Tagline:** Agent-first Web Agency
- **Theme:** Light / Clarity
- **Vibe:** Modern, clean, intelligent, forward-thinking

## Color Palette

### Primary Colors
- **Lumis Light:** `#F5F5F7` (Background - soft off-white)
- **Clarity White:** `#FFFFFF` (Cards, sections)
- **Agent Blue:** `#0066FF` (Primary accent - trust, technology)
- **Deep Space:** `#0A0A0F` (Text, headings)

### Secondary Colors
- **Growth Green:** `#10B981` (Success states, growth metrics)
- **Warning Amber:** `#F59E0B` (Attention, CTAs)
- **Soft Gray:** `#E5E7EB` (Borders, dividers)

### Gradients
```css
/* Hero gradient */
background: linear-gradient(135deg, #F5F5F7 0%, #FFFFFF 100%);

/* Accent gradient */
background: linear-gradient(135deg, #0066FF 0%, #00D4FF 100%);
```

## Typography

### Primary Font: Inter
- Clean, modern, highly readable
- Google Fonts: `https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700`

### Secondary Font: JetBrains Mono
- For code snippets, technical elements
- Google Fonts: `https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;500`

### Font Sizes
- **Hero:** 3.5rem (56px)
- **H1:** 2.5rem (40px)
- **H2:** 1.75rem (28px)
- **H3:** 1.25rem (20px)
- **Body:** 1rem (16px)
- **Small:** 0.875rem (14px)

## Voice & Tone

### Personality
- **Direct** — No fluff, clear value propositions
- **Confident** — Know our worth, state it plainly
- **Helpful** — Genuinely want clients to succeed
- **Modern** — Speak the language of today's web

### Writing Style
- Short sentences
- Active voice
- Concrete benefits over abstract claims
- No corporate jargon

### Example Copy
> "We build websites that work themselves. While you sleep, our agent-first approach ensures your site optimizes, adapts, and grows."

## Logo Usage

### Logo Variants
1. **Primary** — Full logo with wordmark (light backgrounds)
2. **Secondary** — Icon only (favicons, small spaces)
3. **Dark** — Inverted for dark backgrounds

### Clear Space
Maintain minimum clear space equal to the height of the "L" in the logo around all sides.

### Minimum Size
- Full logo: 120px width
- Icon only: 24px × 24px

## Imagery Style
- Clean, minimal compositions
- Soft shadows, subtle depth
- Light color schemes
- Focus on clarity and simplicity

## Brand Don'ts
- ❌ No cluttered layouts
- ❌ No harsh neon colors
- ❌ No stock photo clichés
- ❌ No corporate buzzwords
- ❌ No dark mode (Lumis = light)
