## Style Notes for Image Generation

### Consistent Elements Across All Assets
- **Color palette:** #F5F5F7 (base), #0066FF (accent), #FFFFFF (highlights), #0A0A0F (text)
- **Mood:** Light, clear, modern, intelligent, trustworthy
- **Style:** Minimal, geometric, soft gradients, no harsh shadows
- **Avoid:** Neon colors, dark backgrounds (for primary assets), cluttered compositions, stock photo aesthetics

### Recommended Tools
- **Hero backgrounds:** Midjourney, DALL-E 3, Stable Diffusion XL
- **Logo refinements:** Vectorizer.ai, Figma for final touches
- **Social assets:** Canva for templates, Figma for precision work

### Prompt Tips
- Always specify "no text" for background images
- Include exact hex codes for color consistency
- Request "flat design" or "vector style" for logos
- Specify resolution for each use case
- Use "minimal" and "clean" liberally
